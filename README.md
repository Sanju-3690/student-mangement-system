# Student Management System (Java)

This is a simple Java console application used to manage student records.

## Features
- Add student
- Display students
- Search student by ID
- Delete student

## Concepts Used
- Object Oriented Programming (OOP)
- Classes and Objects
- ArrayList
- Loops and Conditions

## How to Run
1. Compile the program:
   javac Main.java

2. Run the program:
   java Main
